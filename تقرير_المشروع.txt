                    تقرير مشروع معرض السيارات ثلاثي الأبعاد
                         Computer Graphics Project Report
═══════════════════════════════════════════════════════════════════════════════

المقدمة:
---------
مشروع معرض سيارات تفاعلي ثلاثي الأبعاد تم تطويره باستخدام OpenGL و C++.
يتيح المشروع للمستخدم التجول داخل معرض متعدد الطوابق يحتوي على أنواع مختلفة 
من السيارات مع إمكانية التفاعل مع البيئة والسيارات.


═══════════════════════════════════════════════════════════════════════════════
                              المتطلبات المنفذة
═══════════════════════════════════════════════════════════════════════════════

✓ المتطلبات الأساسية:
━━━━━━━━━━━━━━━━━━━━

1. تصميم البيئة الأساسية:
   • معرض من 4 طوابق بتصميم زجاجي حديث
   • بيئة خارجية محيطة بالمعرض (مباني، سماء، أرضيات)
   • مدخل رئيسي بأبواب زجاجية منزلقة
   • مصعد يربط الطوابق الأربعة

2. صالات العرض (4 أنواع):
   
   الطابق الأول - صالة السيارات الرياضية:
   • سيارات: Ferrari F8, Lamborghini, Shelby (موديل 3D)
   • منصات دوارة للسيارات
   • كريستالات ملونة للديكور
   • إضاءة حمراء مميزة
   
   الطابق الثاني - صالة السيارات العائلية:
   • سيارات: Toyota Camry, Honda Accord, Ford Fusion, Mercedes (موديل 3D)
   • منصات دوارة
   • كريستالات خضراء وصفراء
   • إضاءة بيضاء
   
   الطابق الثالث - صالة السيارات الكلاسيكية:
   • سيارات: Ford Mustang, Chevy Camaro, Corvette
   • منصات دوارة
   • كريستالات بنفسجية وزرقاء
   • إضاءة بنفسجية
   
   الطابق الرابع - صالة السيارات الفاخرة:
   • سيارات: Rolls Royce, Bentley, Mercedes S
   • منصات دوارة
   • كريستالات ذهبية
   • إضاءة صفراء ذهبية

3. السيارة الرئيسية (DetailedCar):
   • تصميم كامل من الداخل والخارج
   • مقاعد، مقود، لوحة قيادة
   • أبواب قابلة للفتح (يسار ويمين)
   • إمكانية الدخول والجلوس في مقعد السائق
   • عجلات قابلة للدوران
   • صوت محرك عند التشغيل

4. استخدام Models 3D:
   • Shelby Model (صالة السيارات الرياضية) - ملون أحمر
   • Mercedes Model (صالة السيارات العائلية) - أبيض
   • باقي السيارات مرسومة بالكود (أكثر من 10 سيارات)
   • تم استخدام tiny_obj_loader لتحميل الموديلات

5. البيئة المحيطة:
   • مباني محيطة بالمعرض
   • سماء مع skybox
   • أرضية بخطوط الطريق
   • إضاءة خارجية

6. التحكم والتجول:
   • WASD - الحركة في جميع الاتجاهات
   • Mouse - التحكم بزاوية النظر (360 درجة)
   • Free Roam - التجول الحر في المعرض
   • Driver's Seat - الجلوس في مقعد السائق
   • كاميرا مرتبطة بالماوس ولوحة المفاتيح

7. الشفافية والزجاج:
   • واجهة المعرض الزجاجية الشفافة
   • نوافذ السيارات شفافة
   • أبواب المعرض الزجاجية
   • الكريستالات الشفافة

8. الصدم والحواجز:
   • نظام collision detection للجدران
   • منع المرور عبر المصعد
   • حدود المعرض
   • حدود البيئة الخارجية

9. التفاعل مع الأجزاء:
   • أبواب المعرض (E - فتح/إغلاق)
   • أبواب السيارة (C - فتح/إغلاق)
   • المصعد (1-4 - اختيار الطابق)
   • الإضاءة (L - تشغيل/إيقاف)
   • الوضع الليلي (N - تفعيل/إلغاء)
   • محرك السيارة (X - تشغيل)
   • عجلات السيارة (V - دوران)
   • الدخول للسيارة (R - دخول/خروج)

10. البرمجة غرضية التوجه:
    • Class Showroom - إدارة المعرض
    • Class Camera - التحكم بالكاميرا
    • Class DetailedCar - السيارة التفصيلية
    • Class Model - تحميل الموديلات 3D
    • استخدام OOP principles

11. أنواع الإضاءة:
    • Ambient Light - إضاءة محيطة عامة
    • Directional Light - إضاءة اتجاهية رئيسية
    • Spot Lights - أضواء موجهة لكل سيارة (مختلفة لكل طابق)
    • Point Lights - إضاءة السقف
    • Emission - إضاءة ذاتية للكريستالات


✓ المتطلبات الإضافية:
━━━━━━━━━━━━━━━━━━━━

1. الأصوات:
   • موسيقى محيطة داخل المعرض (ambient-music.wav)
   • صوت فتح الأبواب الزجاجية (glass-door.wav)
   • صوت وصول المصعد (elevator-arrives.wav)
   • صوت فتح باب السيارة (open-car-door.wav)
   • صوت محرك السيارة (car-engine.wav)
   • الأصوات تتغير حسب الموقع (داخل/خارج المعرض)

2. Display Lists:
   • Display List للسيارات البسيطة
   • Display List للكريستالات
   • تحسين الأداء بتخزين الأشكال المتكررة

3. إضافات أخرى:
   • HUD (Heads-Up Display) - عرض معلومات الطابق والتحكم
   • لوحات إرشادية لكل طابق
   • لوحات فنية على الجدران (4 لوحات مختلفة)
   • منصات دوارة للسيارات مع تأثيرات ضوئية
   • خطوط نيون ملونة في كل طابق
   • إضاءة السقف مع تأثيرات
   • نباتات زينة عند المدخل
   • لافتة المعرض الرئيسية
   • Fullscreen mode (F)


═══════════════════════════════════════════════════════════════════════════════
                            التقنيات المستخدمة
═══════════════════════════════════════════════════════════════════════════════

• OpenGL - مكتبة الرسوميات الأساسية
• GLUT - إدارة النوافذ والإدخال
• GLEW - تحميل امتدادات OpenGL
• stb_image - تحميل الصور والخامات
• tiny_obj_loader - تحميل موديلات OBJ
• Windows Multimedia API (MCI) - تشغيل الأصوات
• C++ OOP - البرمجة غرضية التوجه
• Display Lists - تحسين الأداء
• Blending & Transparency - الشفافية والزجاج


═══════════════════════════════════════════════════════════════════════════════
                              أزرار التحكم
═══════════════════════════════════════════════════════════════════════════════

الحركة:
--------
W - التحرك للأمام
S - التحرك للخلف
A - التحرك لليسار
D - التحرك لليمين
Mouse - التحكم بزاوية النظر

التفاعل:
---------
E - فتح/إغلاق أبواب المعرض
C - فتح/إغلاق أبواب السيارة
R - الدخول/الخروج من السيارة
X - تشغيل محرك السيارة
V - تدوير عجلات السيارة

المصعد:
--------
1 - الطابق الأول (السيارات الرياضية)
2 - الطابق الثاني (السيارات العائلية)
3 - الطابق الثالث (السيارات الكلاسيكية)
4 - الطابق الرابع (السيارات الفاخرة)

الإعدادات:
-----------
L - تشغيل/إيقاف الإضاءة
N - الوضع الليلي/النهاري
F - وضع ملء الشاشة
ESC - الخروج من البرنامج

الملفات الرئيسية:
------------------
• main.cpp - نقطة البداية وإدارة البرنامج
• Showroom.cpp/h - إدارة المعرض والصالات (أكثر من 1000 سطر)
• Camera.h - نظام الكاميرا مع collision detection
• DetailedCar.h - السيارة التفصيلية (أكثر من 400 سطر)
• Model.cpp/h - تحميل الموديلات 3D باستخدام tiny_obj_loader
• stb_image.h - تحميل الخامات والصور

المجلدات:
----------
• models/ - موديلات السيارات 3D (Shelby, Mercedes)
• textures/ - الخامات والصور (أرضية، جدران، سقف، لوحات)
• sounds/ - الملفات الصوتية
